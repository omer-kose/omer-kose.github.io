Experience

Currently, I am working as an undergraduate research assistant supervised by Yusuf Sahillioglu and Sinan Kalgkan, funded by TUBITAK.
Supervised by Prof. Rhaleb Zayer.
Worked on mesh flattening under high number of constraints.
Developed a framework in MATLAB where a mesh was mapped onto the Euclidean Plane via various schemes. Implemented different UV solvers ranging from direct construction using optimized angles to several different energy minimizations.
Developed a user interface on MATLAB, enabling users to interactively move, rotate, scale the pre-flattened mesh, and pick constrained vertices for the texture they want to map on the mesh.
Developed a visualizer in C++ with OpenGL to visualize textures on the meshes using the UV coordinates obtained in MATLAB.
Extended the framework to map a face texture on a face mesh.
A pre-trained neural network is used to extract face features from the face texture. The extracted vertices served as constraints of the parameterization problem.
